# 基於RAG的法律案件諮詢系統
## Usage
```bash=
python -m venv venv
pip install -r requirements.txt
source venv/Scripts/activate
python main.py "The situation you encountered"
```
Example:
```bash=
python main.py "我砍伐了自家前面的森林取用木材，但卻被政府提告。"
```
